 AI-Powered Customer Analytics CRM Dashboard

A modern, full-stack data ingestion pipeline built to capture, analyze, and organize client inquiries asynchronously.

-- Tech Stack--
* Backend: Python (Flask Web Engine)
* Database: MySQL Relational Storage
* Frontend: JavaScript (Fetch API), HTML5, Tailwind CSS

-- Core Engineering Features--
* Secure Data Layer: Processes relational data structures utilizing secure prepared statements to completely eliminate SQL Injection threats.
* Asynchronous Lifecycles: Uses client-side AJAX routines to ingest data pipelines seamlessly without forcing manual page updates.
* Modern Interface: Implements an enterprise-grade dark-mode dashboard framework.
